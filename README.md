# Thesisfy

**🏆 2024 한성대학교 SW중심대학 페스티벌 캡스톤 디자인 전시회 우수상 🏆**

> 스마트한 논문 검색의 시작, Thesisfy. <br>
> 대학생과 대학원생을 위한 통합 논문 검색 및 관리 플랫폼입니다.

<br>

## 📌 프로젝트 소개

Thesisfy는 학술 연구의 첫걸음을 떼는 대학생 및 대학원생들이 겪는 자료 검색의 어려움을 해결하기 위해 개발된 iOS 애플리케이션입니다. 여러 학술 정보 사이트를 일일이 방문해야 하는 번거로움을 없애고, 하나의 앱에서 국내 주요 대학의 논문을 통합 검색하고 관리할 수 있는 환경을 제공합니다.

사용자는 Thesisfy를 통해 원하는 키워드로 논문을 검색하고, 관심 있는 논문은 북마크하여 '마이페이지'에서 손쉽게 관리할 수 있습니다. 직관적인 UI와 사용자 경험에 초점을 맞춰 누구나 편리하게 학술 연구에 집중할 수 있도록 돕습니다.

<br>

## ✨ 주요 기능

- **통합 논문 검색**: 국내 주요 대학(서울대, 고려대, 연세대, KAIST 등)의 학술 자료를 한 곳에서 검색합니다.
- **키워드 필터링**: 전공, 학교, 핵심 키워드를 기반으로 정밀하고 빠른 검색 결과를 제공합니다.
- **북마크**: 관심 있거나 중요한 논문을 북마크하여 나만의 논문 컬렉션을 만들 수 있습니다.
- **마이페이지**: 북마크한 논문 목록을 확인하고, 알림 및 프로필을 관리하는 개인화된 공간을 제공합니다.
- **사용자 친화적인 UI/UX**: SwiftUI를 기반으로 제작되어 부드러운 인터랙션과 깔끔하고 직관적인 화면을 제공하여 사용자의 편의성을 극대화했습니다.
